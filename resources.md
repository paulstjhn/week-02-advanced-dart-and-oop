# Resources — Week 2 Overview

A curated list of external materials to support your learning this week.
Each link includes a note on what it covers and when to watch/read it.

---

## 📺 Videos

- [Dart Functions — Full Tutorial](https://www.youtube.com/results?search_query=dart+functions+tutorial)
  Watch before or alongside Topic 01. Covers defining, calling, and parameters.

- [OOP in Dart — Classes and Objects](https://www.youtube.com/results?search_query=dart+classes+objects+OOP)
  Watch before Topic 02. Great visual explanation of the blueprint-vs-object concept.

- [Dart Inheritance and Polymorphism](https://www.youtube.com/results?search_query=dart+inheritance+polymorphism)
  Watch before Topic 04. See real code examples of parent/child classes.

- [Exception Handling in Dart](https://www.youtube.com/results?search_query=dart+exception+handling+try+catch)
  Watch before Topic 06. Short and practical.

---

## 🌐 Official Documentation

- [Dart — Functions](https://dart.dev/language/functions)
  The official reference. Use it to look up specific syntax, not to learn from scratch.

- [Dart — Classes](https://dart.dev/language/classes)
  Covers constructors, fields, methods, and more. Bookmark this.

- [Dart — Extends, Implements, Mixins](https://dart.dev/language/mixins)
  Essential reading before Topic 05.

- [Dart — Error Handling](https://dart.dev/language/error-handling)
  Official guide to exceptions. Read alongside Topic 06 notes.

---

## 🎮 Interactive Practice

- [DartPad](https://dartpad.dev)
  Run any Dart code in your browser — no setup needed. Use this to experiment
  with every example in the notes before running it locally.

---

## 📖 Further Reading (Optional)

- [OOP Concepts Explained Simply](https://www.freecodecamp.org/news/object-oriented-programming-concepts-21bb035f7260/)
  Language-agnostic explanation of OOP principles. Great if the concepts feel abstract.

- [SOLID Principles — A Gentle Introduction](https://www.freecodecamp.org/news/solid-principles-explained-in-plain-english/)
  Not required this week, but worth reading once inheritance clicks. This is where
  OOP gets truly powerful.
