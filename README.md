## 🎯 Learning Objectives

By the end of this week, you should be able to:

- [ ] Define and call functions with positional, named, and optional parameters
- [ ] Read and parse keyboard user input from the console using `stdin`
- [ ] Create classes with custom constructors, member fields, and behaviors
- [ ] Apply encapsulation principles using private fields, getters, and setters with validation
- [ ] Use inheritance to reuse code and build subclass hierarchies
- [ ] Apply polymorphism by overriding parent class methods
- [ ] Implement abstract classes and interfaces to enforce API designs
- [ ] Use mixins to share behaviors across independent classes
- [ ] Handle exceptions and errors gracefully using `try/catch/finally` blocks
