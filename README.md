# Week 3 — Functions, Classes, and Object-Oriented Programming

## Learning Objectives

By the end of this week you should be able to:

- [ ] Define and call functions with parameters and return values
- [ ] Read and handle user input from the console
- [ ] Create classes with constructors, fields, and methods
- [ ] Apply encapsulation using private fields, getters, and setters
- [ ] Use inheritance to build on existing classes
- [ ] Apply polymorphism through method overriding
- [ ] Implement abstract classes and interfaces
- [ ] Use mixins to share behaviour across unrelated classes
- [ ] Handle errors gracefully using try/catch/finally
